Bu uygulamada bison ve flex kullanılmıştır.
Windows ortamında flex ve bison version 2.5.17 ve üstü (bison 3.3.1, flex 2.6.4) gereklidir.
bison_flex klasörü içerisinde bulunmaktadır.Aşağıdaki linkten son sürümü indirilebilir.
https://github.com/lexxmark/winflexbison/releases
(g++ gereklidir.https://www.youtube.com/watch?v=sXW2VLrQ3Bs)
Windows ortamında run.bat'ı çalıştırarak kısa yoldan derleyebilirsiniz.
run.bat geliştirme aşamasında hızlı derleyebilmek için yazılmıştır.

Linux ortamı için (flex bison g++)
sudo apt-get update
sudo apt install g++
sudo apt install build-essential
sudo apt-get install flex
sudo apt-get install bison
which flex
which bison 

vscode flex_bison eklentisi için;
